# Lpgrid

Last Pass Grid reader

Last pass introduced another layer of protection with Grid card. 
This small Ruby program is here to help you read you grid card
information more easily.

## Installation

    gem install 'lpgrid'

## Usage

    bin/lpgrid

## Tests

To run tests trigger

    rake
